## Preview

<AttachmentsDemo client:only="solid" />

<Aside type="tip" title="When to use">
Show files a user attached (or the assistant produced) **anywhere in a chat** — the composer, a message body, a sources rail. Set `variant` to fit the space: `grid` for a gallery, `inline` for a compact chip row, `list` for a detailed column.
</Aside>

## Usage

Set `items` as a JS **property**; configure presentation with **attributes** (`variant`, `hover-card`, `removable`); listen for the `kc-remove` event. The **Code** panel in the Preview above generates the exact snippet for your framework as you toggle the controls.

## Examples

Each example is live — expand **Code** to copy the exact snippet for your framework.

### Grid

`variant="grid"` — square image tiles for a gallery.

<AttachmentsExample client:visible variant="grid" images />

### Inline

`variant="inline"` — a compact icon + label chip per file, for a composer or message body.

<AttachmentsExample client:visible variant="inline" />

### List

`variant="list"` — full-width rows with a preview thumbnail, filename and media type.

<AttachmentsExample client:visible variant="list" />

### Removable

Add `removable` for a per-item remove button (the **×**); it fires the `kc-remove` event.

<AttachmentsExample client:visible variant="list" removable />

### Hover preview

Add `hover-card` to reveal each item's details on hover — useful on `grid`, where the tile shows only the image.
