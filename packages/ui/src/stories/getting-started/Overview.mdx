# Overview

This Storybook is the dev surface for AI/UI: the SolidJS layer under the `kai-*` web components. The headless primitives, the UI primitives, the feature components, and the custom-element facades, each with live stories and controls.

Consumer docs (install, the chat app, the full element API) live at [ui.kitn.ai](https://ui.kitn.ai). Use this when you're composing the kit directly in Solid, or working on it.

## Sections

- **Getting Started**: how the kit is built, running it locally, adding or changing a component.
- **Components**: a story for every primitive and feature component.
- **Labs**: compositions prototyped against real `kai-*` elements before they graduate to the docs.

- [How it's built](?path=/docs/getting-started-how-it-s-built--docs): the four layers.
