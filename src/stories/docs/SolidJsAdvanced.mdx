# Components

The SolidJS layer the `kai-*` web components compose from. Every primitive and feature component, with controls and source. The full element API and framework usage are at [ui.kitn.ai](https://ui.kitn.ai).

## Two tiers

- **Elements**: feature components (`Message`, `Tool`, `Reasoning`, `Composer`, `Artifact`, …), each mapping to a `kai-*` element.
- **Primitives**: the accessible base components (`Button`, `Tooltip`, `Dropdown`, `Switch`, …) they're built on. No third-party UI deps.

## Each story

- **Demo**: the live component, with controls.
- **Actions**: events stream here as you interact.
- **Code**: the story's SolidJS source.

Theme everything with the design tokens in [Token Reference](?path=/docs/getting-started-token-reference--token-reference).
