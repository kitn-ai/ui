## Usage

Register the custom elements once, then set the `code` **property** and the `language` **attribute**:

```html
<script type="module" src="https://cdn.jsdelivr.net/npm/@kitn.ai/chat/dist/kitn-chat.es.js"></script>

<kc-code-block id="cb" language="ts"></kc-code-block>

<script type="module">
  const cb = document.getElementById('cb');
  cb.code = `export function greet(name: string) {
  return \`Hello, \${name}!\`;
}`;
</script>
```

A few things worth knowing:

- **`code` is a JS property**, not an attribute. Assign it with `el.code = '...'` тАФ trying to set it as an HTML attribute will not work for multiline strings.
- **`language`** is a plain HTML attribute (defaults to `tsx`). Pass any [Shiki grammar name](https://shiki.style/languages), such as `ts`, `python`, `bash`, `json`, or `html`.
- **`code-theme`** is also an attribute. It defaults to `github-dark-dimmed`. Any [Shiki theme](https://shiki.style/themes) works тАФ try `github-light`, `nord`, or `dracula`.
- Set **`code-highlight="false"`** to skip Shiki entirely and render plain monospace text тАФ useful for raw log output or untokenisable content.
- The **Code** panel in the Preview above generates the exact framework snippet as you adjust the controls.
