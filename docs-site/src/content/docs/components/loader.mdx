<Aside type="tip" title="When to use">
Reach for `<kc-loader>` whenever you need a **"working" indicator outside the chat thread** — inside a send button, in a toolbar corner, or as a placeholder while a panel loads. In SolidJS, use the `Loader` primitive directly for finer composition control.
</Aside>

## Usage

Register once — `import '@kitn.ai/chat/elements'` — then drop the element anywhere:

```html
<kc-loader variant="dots" size="md"></kc-loader>
```

All three props are plain **HTML attributes**, so they work in any framework without a binding layer. Pick a `variant` to match the context (spinning circle for a generic wait, `wave` or `bars` for audio/processing, `text-shimmer` for an AI "thinking" state). Use `size` to scale with the surrounding UI (`sm` inside a button, `lg` as a panel placeholder). For the text-based variants (`text-blink`, `text-shimmer`, `loading-dots`), set `text` to the label you want animated — for example `text="Thinking…"`.

The **Code** panel in the Preview above generates the exact framework snippet as you adjust the controls.
