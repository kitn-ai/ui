<Aside type="tip" title="When to use">
Reach for `<kc-file-tree>` whenever you need a sidebar-style file explorer — inside `<kc-artifact>`'s Code tab, a `<kc-resizable>` panel, a project-level attachment browser, or any file-picker surface. ## Usage

Register the custom elements once, then set the `files` **JS property** (not an attribute — it's an array):

```html
<script type="module" src="https://cdn.jsdelivr.net/npm/@kitn.ai/chat/dist/kitn-chat.es.js"></script>

<kc-file-tree style="display:block;height:320px"></kc-file-tree>

<script type="module">
  import '@kitn.ai/chat/elements'; // registers the custom elements

  const tree = document.querySelector('kc-file-tree');

  // `files` is a JS property (array). Folders are derived from `/` in each path.
  tree.files = [
    { path: 'index.html', type: 'html' },
    { path: 'src/app.ts', type: 'other' },
    { path: 'src/lib/format.ts', type: 'other' },
    { path: 'assets/logo.svg', type: 'image' },
  ];

  tree.setAttribute('active-file', 'src/app.ts');
  tree.addEventListener('kc-select', (e) => console.log('selected', e.detail.path));
</script>
```

A few things worth knowing:

- **`files` is a JS property** — assign the array with `el.files = [...]`. Each entry needs a `path` string; `type` (`'html' | 'pdf' | 'image' | 'other'`), `language`, `url`, and `code` are optional and pick the file-row icon or source.
- **Folders are derived** — no explicit folder entries are needed. Any `/` in a path creates the intermediate folder nodes automatically.
- **`active-file`** is a plain attribute (kebab-case). Set it to highlight a row; update it in response to `kc-select` to keep the selection in sync.
- **`default-expanded`** is a JS property (`el.defaultExpanded = ['src', 'src/lib']`). Omit it to start with all folders open; pass an array to restrict which folders start expanded. Folder paths use the same slash-delimited format as `files` entries.
- **Keyboard navigation** — the element is a proper ARIA `tree`. Arrow keys move focus; Enter or Space selects a file or toggles a folder; Home/End jump to the first/last visible row.
- **Events** — `kc-select` fires on file selection with `{ path: string }` in `detail`. See the **Events** table below and the **Code** panel above for per-framework listener syntax.
