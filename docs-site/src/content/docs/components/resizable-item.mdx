## Preview

<Playground client:only="solid" tag="kai-resizable-item" />

<Aside type="tip" title="When to use">
Use `<kai-resizable-item>` as a direct child of `<kai-resizable>` to define each panel. Give panels with a natural default width an explicit `size` (e.g. a sidebar at `"260px"`); leave `size` off the main content area so it fills remaining space. Use `locked` for chrome panels that must never resize, and toggle `hidden` to collapse a panel on demand.
</Aside>

## Usage

`<kai-resizable-item>` has no standalone rendering ‚Äî always place it inside `<kai-resizable>`:

```html
<kai-resizable orientation="horizontal" style="display:block;height:400px">
  <kai-resizable-item size="260px" min="160px" locked>
    <!-- sidebar content -->
  </kai-resizable-item>
  <kai-resizable-item>
    <!-- main content fills remaining space -->
  </kai-resizable-item>
  <kai-resizable-item size="35%" min="200px">
    <!-- preview / inspector pane -->
  </kai-resizable-item>
</kai-resizable>

<script type="module">
  import '@kitn.ai/ui/elements';
</script>
```

- **`size`** ‚Äî initial main-axis size: `"280px"` or `"25%"`. Omit for a flexible panel (`flex: 1 1 0%`).
- **`min` / `max`** ‚Äî drag bounds in px or %, e.g. `min="120px"`, `max="50%"`.
- **`locked`** ‚Äî fixes the panel; adjacent dividers become non-draggable separators.
- **`hidden`** ‚Äî removes the panel and its divider; remaining panels reflow. Toggle via JS to collapse/restore.

Resize events (`kai-change`, `kai-maximize-change`) fire on the parent `<kai-resizable>`, not the item. See the [Resizable](/components/resizable) page for the full event reference.

## Examples

### Flexible Panel

No `size` ‚Äî stretches to fill whatever space siblings leave. Right choice for the primary content area.

<Example client:visible tag="kai-resizable-item" data="flexible" />

### Sized Panel

`size="260px"` sets the initial width. The user can still drag to resize unless `locked` is also set.

<Example client:visible tag="kai-resizable-item" config={{ size: '260px', min: '140px', max: '50%' }} data="sized" />

### Locked Panel

`locked` fixes the panel at its `size` ‚Äî adjacent dividers become non-draggable. Use for nav sidebars and tool panels.

<Example client:visible tag="kai-resizable-item" config={{ size: '240px', locked: true }} data="locked" />
