## Preview

<Aside type="tip" title="When to use">
Reach for `<kc-image>` when you receive raw image data from a model or API response and need to display it inline — inside a message bubble, a card body, or an artifact preview — without uploading it first. In SolidJS, compose the `Image` primitive directly for finer control. The element is `display: inline-block` and the image fills to the parent container's width, so constrain width on the parent when needed.
</Aside>

## Usage

Register the custom elements once, then assign either `base64` (paired with `media-type`) or `bytes` as a property:

```html
<script type="module" src="https://cdn.jsdelivr.net/npm/@kitn.ai/chat/dist/kitn-chat.es.js"></script>

<kc-image id="img" media-type="image/png" alt="A generated chart"></kc-image>

<script type="module">
  import '@kitn.ai/chat/elements';   // registers the custom elements

  const img = document.getElementById('img');

  // Option A — base64 string (attribute-compatible):
  img.base64 = '<...base64 encoded image data...>';

  // Option B — raw bytes as a property:
  // img.bytes = new Uint8Array([...]);
</script>
```

A few things worth knowing:

- **`base64`** is a scalar attribute — you can set it as `el.base64 = '...'` or as the `base64` HTML attribute. Pair it with the `media-type` attribute so the browser decodes it correctly.
- **`bytes`** is a **JS property** (non-scalar). It accepts a `Uint8Array` directly — set it as `el.bytes = new Uint8Array([...])`. You do not need `media-type` when using raw bytes if the format is detectable, but providing it is recommended.
- **`media-type`** defaults to `image/png`. Override it for SVG (`image/svg+xml`), JPEG (`image/jpeg`), or any other MIME type.
- **`alt`** defaults to an empty string. Always provide meaningful alt text for accessibility — screen readers announce it in place of the image.
- A **skeleton placeholder** is shown while the image resolves, so layout does not shift unexpectedly on first paint.
- The **Code** panel in the Preview above generates the exact framework snippet as you adjust the controls.
