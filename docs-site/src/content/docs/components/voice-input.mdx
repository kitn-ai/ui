<Aside type="tip" title="When to use">
Reach for `<kc-voice-input>` when you want to add microphone-based dictation to a prompt input, search bar, or form field in **any framework** — Vue, Angular, Svelte, React, or plain HTML. In SolidJS, compose `VoiceInput` from `@kitn.ai/chat` directly for tighter reactive control. The element renders as an inline mic button sized to fit any toolbar.

## Usage

Register the element once, then wire up the `transcribe` **function property** and listen for the `kc-transcription` event:

```html
<script type="module" src="https://cdn.jsdelivr.net/npm/@kitn.ai/chat/dist/kitn-chat.es.js"></script>

<kc-voice-input id="voice"></kc-voice-input>

<script type="module">
  const voice = document.getElementById('voice');

  // transcribe is a FUNCTION property — assign it imperatively after the element
  // is in the DOM. It receives the recorded Blob and must return a Promise<string>.
  voice.transcribe = async (blob) => {
    const form = new FormData();
    form.append('audio', blob, 'recording.webm');
    const res = await fetch('/api/transcribe', { method: 'POST', body: form });
    const { text } = await res.json();
    return text;
  };

  // Fired when transcription completes — wire the text to your input.
  voice.addEventListener('kc-transcription', (e) => {
    document.getElementById('prompt').value = e.detail.text;
  });
</script>
```

A few things worth knowing:

- **`transcribe` is a JS property, not an attribute.** You must assign it with `el.transcribe = async blob => '...'` after the element is in the DOM. It is a function-valued callback — the element calls it with the recorded `Blob` and expects a `Promise<string>` back.
- **Two event paths** — if you prefer to transcribe outside the element entirely, skip setting `transcribe` and listen for `kc-audio-captured` instead. The event fires with `{ blob: Blob }` immediately after recording stops. The `kc-transcription` event fires only when the `transcribe` function resolves.
- **`disabled`** is a plain boolean attribute/property. Set it when recording is not yet allowed (e.g. while the assistant is responding).
- **Registration** — `import '@kitn.ai/chat/elements'` is the recommended one-liner; the CDN `<script>` above is a zero-install alternative for plain HTML pages.
- Open the **Code** panel in the Preview above to copy the exact framework snippet.
